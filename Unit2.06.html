Unit 3.6 - Error Detection

1)(POGIL) Describe an algorithm for identifying the card that was flipped.

 The algorithm for identifying the card that was flippedis going by making a 5*5 array of cards,then add an extra row and column for error detection.The extra column should make all the column and row have 4 cards flips or 2 cards flip,so if other people flip the cards,then the column and row who get 5 or 3 cards flip would be the error.

2)(POGIL) The card "trick" shows that it is always possible to identify the card that was flipped as long as only one card was flipped. Would it be possible always to determine if an error occurred if two cards were flipped?

 It wouldn't be possible always to determine if an error occurred if two cards were flipped when the cards are flip in the same column or row.But it can still determine if two cards flips in different row and column.
